🛒 Projeto Loja Virtual

Este projeto é uma página de loja virtual desenvolvida por mim para praticar e aprender desenvolvimento web utilizando HTML, CSS e JavaScript.

🚀 Sobre o projeto

A ideia do projeto foi criar uma loja simples, moderna e responsiva, aplicando conceitos básicos de front-end como:

Estruturação com HTML
Estilização com CSS
Interatividade com JavaScript
Layout responsivo
Cards de produtos
Botões interativos
Organização visual de uma loja online
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript
📚 Objetivo

Esse projeto faz parte do meu aprendizado como desenvolvedor iniciante, buscando melhorar minhas habilidades na criação de interfaces web e funcionalidades interativas.
